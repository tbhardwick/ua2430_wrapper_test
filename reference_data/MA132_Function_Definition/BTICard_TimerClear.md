# **TimerClear**

```
VOID BTICard_TimerClear
```

```
(
      HCORE hCore //Core handle
)
```
## **RETURNS**

None.

## **DESCRIPTION**

Clears the Device timer to zero.

#### **DEVICE DEPENDENCY**

Applies to all Devices.
